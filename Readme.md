# 42sh

## Projet

Le but était de réaliser un shell complètement fonctionnel, avec la gestion de commande, d'historique, d'autocomplétion, des alias, des backticks et bien d'autres !

## Utiliser le shell

### Prérequis
    - C

```bash
make
./42sh
```
OU
```bash
./run.sh
./42sh
```
